# 🌍 Pro Universal Business Analytics Dashboard

The **Pro Universal Business Analytics Dashboard** is an interactive tool that lets users upload any CSV or Excel dataset, filter and analyze numeric, categorical, and date data, visualize KPIs and charts, and gain automated insights — all without writing code.

---

## Features

- Automatically detects numeric, categorical, and date columns.  
- Filter data by ranges, categories, and dates.  
- View KPIs: sum, average, min, max.  
- Interactive visualizations: bar charts, pie charts, trend lines.  
- Automated insights highlighting top products, customers, or categories.  
- Download filtered data as CSV.  
- Dark mode support for comfortable viewing.

---

## Built With

- Python  
- Streamlit  
- Plotly Express  
- Pandas  

---

## Getting Started

1. Clone the repository:  
```bash
git clone https://github.com/yourusername/pro-universal-analytics-dashboard.git
cd pro-universal-analytics-dashboard
```
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Run the dashboard:
```bash
streamlit run app.py
```
4. Upload your CSV or Excel file and explore your data.
